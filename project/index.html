LiquidML

LiquidML is a distributed scalable machine learning platform. It aims to abstract over the highly parallelized operations by providing convenient filtering and mapping interfaces.
Rust Data Analysis Machine Learning Random Forest

Sorer

Sorer is a high performance multi-threaded file parser designed to read and store the data from a CSV file. It uses a probablistic algorithm to infer the schema and then stores it in memory in a columnar format. It was originally designed to work with LiquidML.
Rust Parsing CSV Multi Threaded

Programming Language

An interpreter for a lisp like language with support for static type checking, mutation, and functions as first class citizens. It uses a sexpr reader and then parses it to an abstract syntax tree. The interpreter then uses consistent scoping rules to interpret and resolve the program.
Rust Programming Languages Parsing Interpreter

GShell

A system shell with support for common bash operators. This was a reimagining of a system shell I wrote at school. This was my first foray into rust programming, I have been captivated by it since.
Rust Shell Systems Parsing

NUFS

A ext like file system built on the fuse API. It is fully built out and supports the entire standard file system operations. The filesystem supports storing files of arbitrary sizes (not the block size), symlinks, arbitrary nesting of directories.
